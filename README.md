# fotazapp

A minimal project (inspired by instagram) in Django.

![](https://i.imgur.com/4VotR0d.png)

## Dependencies
Python 3
Django

## Usage
```shell
python3 -m venv .env
source .env/bin/activate
```

```python
python manage.py makemigrations
python manage.py migrate
python manage.py runserver
```
